## Session 2 — 2026-05-17 (Full Audit and Correction)

### Root Cause Analysis

**Critical bug: SampEn as primary discriminant was mathematically incorrect.**

`antropy.sample_entropy()` computes template matches in absolute Chebyshev
distance space. For a signal with mean=60s and std=10s (30% jitter), it
returns SampEn≈2.2, which is ABOVE the threshold of 1.8 → detection MISSED.
For a human-traffic Poisson signal with mean=2s and std=2s, SampEn≈1.5 →
BELOW threshold → would be flagged as C2. The feature was backwards.

**Correct primary discriminant: Coefficient of Variation (CV = std/mean).**

CV is scale-invariant. For a uniform jitter beaconer:
  - CV = J/√3 ≤ 0.5/√3 ≈ 0.289 (for any reliable beaconer with J ≤ 50%)
  - Human traffic (exponential/Poisson): CV ≈ 1.0

This is why the CV-based formal bound in §IV of the paper is provably correct
and the SampEn-based bound needed to be secondary/informational only.
